## Base Input Text
---

- **Usage**:

Form input that expects a user to type.

- **When to use**:

   - Publicly available
   - Not a fine grain component
   - 

- **Props**:

   - `text` - string
   - `type` - string, optional value: `"tip"|"warn"|"error"`, defaults to `"tip"`.
   - `vertical` - string, optional value: `"top"|"middle"`, defaults to `"top"`.
